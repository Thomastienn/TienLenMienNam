# IN DEVELOPMENT

## How to use
1. Navigate to your download folder by `cd <path to your download folder>`
2. `git clone https://github.com/Thomastienn/TienLenMienNam`
3. Open your IDE
4. Import everything in the folder TienLenMienNam
5. Change images directory (constant: imgDir) if there is a top-level folder (Details in src code)
6. Run and experience

## How to contribute
1. Navigate to the project folder `cd <path to the project>`
2. Create remote `git remote add bigtwo https://github.com/Thomastienn/TienLenMienNam`
3. Update with the current version `git fetch bigtwo main`
4. Update the src code
5. Push back to git `git push bigtwo main`
