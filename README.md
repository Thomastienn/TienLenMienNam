# TIEN LEN MIEN NAM (Big Two)

## How to use
1. Navigate to your download folder by `cd <path to your download folder>`
2. `git clone https://github.com/Thomastienn/TienLenMienNam`
3. Open your IDE
4. Import everything in the folder TienLenMienNam
5. Change images directory (constant: imgDir) if there is a top-level folder (Details in src code)
6. Run and experience

## How to contribute
1. Fork this repo using the `Fork` button
2. Navigate to the project folder `cd <path to the project>`
3. `git clone https://github.com/<your-github-username>/<your-repo-name>`
4. Updating the src code
5. Create a new branch `git checkout -b <new-branch-name>`
6. Staging all the codes `git add .`
7. Commit all the changes `git commit -a -m "<your messages>"`
8. Push back to your github `git push origin <your-branch-name>`
9. Create a pull request -> Click the `New pull request` button on github
